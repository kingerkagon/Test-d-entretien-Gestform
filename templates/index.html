Traitement de la liste aléatoire

Soit une liste aléatoire de nombres entiers, compris entre -1000 et 1000.
Pour chaque nombre n de liste, on veut effectuer les opérations suivantes :

  1. Si le nombre est divisible par 3 : on affiche Geste
  2. Si le nombre est divisible par 5 : on affiche Forme
  3. Si le nombre est divisible par 3 et par 5 : on affiche Gestform
  4. Sinon : on affiche le nombre n
